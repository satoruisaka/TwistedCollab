TwistedCollab — Quick Guide

🔍 Search Tab

  • Data Source: Select which sources to include in RAG retrieval. When none are selected the query goes directly to the LLM chat.
  • Upload Files: Add documents to your user_uploads collection.
  • LLM Settings: Configure model, temperature and token limits (collapsed by default).
  • Distortion: Set Mode to anything other than Off to enable TwistedPair rhetorical distortion. Default is Off.
  • Quick Notes: Mini notepad on the right for jotting during research.

📝 Notes Tab

  • Markdown Editor with GFM support and live preview.
  • Preview Modes: Edit / Split (side-by-side) / Preview.
  • Auto-save: Notes saved every 30 seconds.

💬 Sessions Tab

  • Browse all past conversations grouped by date.
  • Click any session to resume it.

⌨️ Keyboard Shortcuts

  • Ctrl+S — Save current note
  • Ctrl+N — New note (Notes tab)
  • Ctrl+O — Open note (Notes tab)
  • Ctrl+Shift+S — Download note
  • Ctrl+E — Cycle preview modes (Notes tab)

🔌 Status Indicators

  • Green (pulsing): Service online
  • Red: Service offline or unreachable

🎸 Distortion Modes

  • Off — No distortion (default)
  • Echo-er — Amplify / positive framing
  • Invert-er — Negate / challenge assumptions
  • What-if-er — Explore alternatives
  • So-what-er — Demand implications
  • Cucumb-er — Academic / analytical
  • Archiv-er — Historical context
